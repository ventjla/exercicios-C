# 💻 Exercícios de Desenvolvimento de Sistemas

Este repositório foi criado para centralizar todos os exercícios, desafios e projetos práticos desenvolvidos ao longo do curso de Desenvolvimento de Sistemas. O objetivo é registrar minha evolução técnica e servir como material de consulta rápida.

---

## 🛠️ Tecnologias Utilizadas

Aqui estão as principais linguagens, frameworks e ferramentas que utilizo nos exercícios:

* **Linguagens:** Java, JavaScript, Python, SQL
* **Front-end:** HTML5, CSS3, Bootstrap
* **Banco de Dados:** MySQL, PostgreSQL
* **Ferramentas:** Git, VS Code, IntelliJ IDEA

---

## 📁 Estrutura do Repositório

O projeto está organizado por módulos ou disciplinas para facilitar a navegação:

```text
├── 01-logica-programacao/   # Algoritmos básicos e estruturas condicionais/repetição
├── 02-banco-de-dados/       # Modelagem, scripts DDL e consultas DML (SQL)
├── 03-front-end/            # Páginas web estáticas e responsivas
├── 04-programacao-web/      # Criação de APIs, rotas e integração com banco de dados
└── README.md                # Documentação do repositório
```

## 🚀 Como Executar os Projetos

### Pré-requisitos
Antes de começar, você vai precisar ter instalado em sua máquina:
* [Git](https://git-scm.com)
* Um editor de código (como o [VS Code](https://visualstudio.com))
* O ambiente de execução correspondente à linguagem da pasta (ex: Node.js, JDK).

### Passo a Passo

1. Clone este repositório para a sua máquina local:
   ```bash
   git clone https://github.com
   ```
2. Entre na pasta do exercício desejado:
   ```bash
   cd nome-da-pasta
   ```
3. Siga as instruções específicas de execução contidas no arquivo README interno da pasta (se houver).
